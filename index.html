Hong Kong Phantom Buses

This is a proof of concept on consuming and restructuring the bus ETA data provided by Hong Kong Governement's Open Data API. I used SpringBoot to develop a backend service that request and consume real time data from the API. Then, I created a frontend to further consume the processed information using React, Redux and React Hook.
